JS: Question 1:

This text should be dark blue.
1. Explain why the JavaScript code in the <head> element is failing to achieve this.
2. Edit the JavaScript code to achieve this, leaving it in the <head>.
The code is wrong in many different ways. The most obvious error is the .text-color property. Apart that there is no such property, dashes in js are not sytactically correct so they will always give you an error. The "text-color" property is actually just 'color'. Secondly, getElementsByClassName will return an array - even if there is one item you still have to stipulate [0] after it. Finally, even if we fix the syntax: document.getElementsByClassName("blue")[0].style.color = "darkblue"; class blue is not defined since the span with class blue is way down the file (JS executes here as single-threaded). In order to fix the code, we can wrap the code into an event listener function that will get executed on load. Now through Javascript's hoisting (adding functions and variable declarations to the memory first prior to executing the actual code) class blue will be available.
